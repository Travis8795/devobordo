The board is compatible with the official Pico SDK.
[Setup guide](https://datasheets.raspberrypi.com/pico/getting-started-with-pico.pdf)

| Function   | GPIO                     |
| ---------- | ------------------------ |
| Status LED | GPIO25                   |
| USB        | GPIO24 / GPIO25 internal |
| SWD Debug  | SWCLK / SWDIO            |
| UART       | GPIO0 / GPIO1            |
| I2C        | GPIO4 / GPIO5            |
| SPI        | GPIO16–19                |
